This web page was developed using React, HTML, CSS, and JavaScript, integrating the Spotify API.

It is a unique web with a design based on spotify-tui and focused on user experience.

The application allows users to explore and play music through their playlist. Initially, it was a simple project, but during the development, it expanded to include more features, such as playing songs, switching between private and public playlists, etc.
