Este proyecto es una recreación de la popular plataforma Discord, desarrollado para explorar tecnologías web modernas y practicar integraciones de API. El objetivo fue construir una aplicación funcional con comunicación en tiempo real como autenticación segura, gestión de servidores y canales, chat en tiempo real y llamadas de voz y video, mientras se aprendía a combinar múltiples servicios en un único producto integrado.
