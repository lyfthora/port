Github

El clone de Animeflv fue inicialmente diseñado solo con HTML y CSS durante mi práctica. Posteriormente decidí adaptarlo al framework de React para mejorar mis habilidades en desarrollo frontend.

Esta transformación hizo que el proyecto fuera más dinámico, interactivo, y totalmente responsivo, mejorando su adaptabilidad en diferentes dispositivos.

Lo que comenzó como un simple proyecto frontend evolucionó con React, permitiendo el uso de componentes reutilizables y características avanzadas de frontend, llevando a una experiencia de usuario más inteligente.
