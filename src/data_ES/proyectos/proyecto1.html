Este sitio web fue construido utilizando solo HTML, CSS, JavaScript y PHP, con ninguna otra dependencia.

También es un sitio web dinámico que es completamente responsivo.

La página está inspirada en Serial Experiments Lain. Inicialmente, solo se iba a utilizar HTML, CSS y JavaScript. Sin embargo, mientras la desarrollaba, decidí incorporar PHP para añadir más funcionalidad.
