ChatBot

[Construido en 2024]

HTML CSS JavaScript Vue.js
Github

El chatbot fue creado durante mi práctica, ya que tenía que ser un proyecto relacionado con mi trabajo en IA. Elegí desarrollar un chatbot para una institución.

Fue construido para ser dinámico, interactivo y totalmente responsivo para todos los dispositivos.

Inicialmente diseñado como un chatbot simple basado en texto, ha evolucionado para incluir reconocimiento de voz e integración con la API de OpenAI, añadiendo respuestas inteligentes y características avanzadas.
