[Construido en 2024]

Github

Esta pagina web fue desarrollada utilizando React, HTML, CSS y JavaScript, integrando la API de Spotify.

Es una web unica con un diseño basado en spotify-tui y centrada en la experiencia del usuario.

La aplicación permite a los usuarios explorar y reproducir música, atraves de su playlist. Inicialmente era un proyecto simple, pero durante el desarrollo se expandió para incluir mas funcionalidades como poder reproducir las canciones,poder cambiar entre las playlist privadas y publicas,etc.
